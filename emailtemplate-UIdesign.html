Responsive Email Template UI Design


UI Design challenge for Google's recruitment email templates for UX Research

Project Overview

Project Type: UI Design Challenge for Employer (non-confidential)
Duration: 1 day


Tools Used

Sketch, Adobe XD


Redesigned an exciting, responsive email template that catches the reader's eye, while making it obvious who the email came from.

Problems before redesign:

  • Extremely text-heavy, all text is in the same size
  • All black-and-white
  • Spacing does not separate important messages from unimportant subtext.
  • Not mobile-friendly

Overall Image

Email is bland and lacks in engagement, resulting in response rate lower than 10%.

Changes made in redesign:

  • Add color and depth to email to catch the reader's eye
  • Use CTA buttons and illustrations for engagement.
  • Separate main message from unimportant text.
  • Keep color consistent throughout email, use different color per email
